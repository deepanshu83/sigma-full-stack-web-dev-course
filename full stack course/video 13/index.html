### Currency Symbols
- `$` → `$` (dollar)
- `€` → `€` (euro)
- `£` → `£` (pound)
- `¥` → `¥` (yen)

### Mathematical Symbols
- `+` → `+` (plus)
- `−` → `−` (minus)
- `×` → `×` (times)
- `÷` → `÷` (divide)
- `=` → `=` (equals)
- `≠` → `≠` (not equal to)
- `≤` → `≤` (less than or equal to)
- `≥` → `≥` (greater than or equal to)

### Other Symbols
- `©` → `©` (copyright)
- `®` → `®` (registered trademark)
- `™` → `™` (trademark)
- `•` → `•` (bullet)
- `…` → `…` (ellipsis)

### Accented Letters
- `á` → `á` (a with acute accent)
- `é` → `é` (e with acute accent)
- `í` → `í` (i with acute accent)
- `ó` → `ó` (o with acute accent)
- `ú` → `ú` (u with acute accent)
- `ñ` → `ñ` (n with tilde)
- `ü` → `ü` (u with diaeresis)

### Other Characters
- `–` → `–` (en dash)
- `—` → `—` (em dash)
- `“` → `“` (left double quote)
- `”` → `”` (right double quote)
- `‘` → `‘` (left single quote)
- `’` → `’` (right single quote)

These are just a few of the many HTML entities available. They help
 represent characters that may otherwise conflict with HTML syntax 
 or are not easily typed directly into a document.
